# customer_orders

A simple go REST service to show implementation of PostgreSQL, CI/CD, Open ID Connect, testing, deployment

## Setup

### Init

`$ go mod  tidy`

### Swagger

`$ make swaginstall`
`$ make swag`

### Server

`$ make make server`

### Testing and coverage

`$ make test`
`$ make testcover`
`$ make testview`
